# Markdown Maker

Offline, installable PWA — convert PDF, image, or HTML to clean Markdown for cheaper AI context.

**Try it:** [markdown-maker.pages.dev](https://markdown-maker.pages.dev) · Coming soon: [markdown-maker.levelone.co.za](https://markdown-maker.levelone.co.za)

**Install:** Open in Chrome or Safari → address-bar install icon (desktop) or Share → Add to Home Screen (mobile).
